# 异步粒子

- 中文|[English](./README.md)
- [下载](https://github.com/Harveykang/AsyncParticles/releases)

## 特性

- Minecraft
  - 异步粒子刻/渲染
  - 异步客户端方块实体刻
- 瓦尔基里天空/机械动力 + Pretty Rain/粒子雨
  - 天气粒子不会穿过船只/动态结构
- Pretty Rain/粒子雨/Effectual/Particular
  - 异步粒子生成

## 推荐模组

- ModernFix
- Sodium/Embeddium
- Flerovium
- 星光/ScalableLux
