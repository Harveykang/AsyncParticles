# AsyncParticles

- [Downloads](https://github.com/Harveykang/AsyncParticles/releases)

## Features

- Minecraft
  - Async particle ticking/rendering(buffer filling).  
  - Async client block entity ticking.
- Valkyrien Skies + Pretty Rain/Particle Rain
  - Weather particles no longer pass through ships.
- Pretty Rain/Particle Rain
  - Async particle gen.
- Effectual
  - Async particle gen.
- Particular
  - Async particle gen.

## Mods Recommended

- ModernFix
- Sodium/Embeddium
- Flerovium
- Startlight
